Java API文档

解释：API（Application Programming Interface，“应用”“编程”“接口”）

通过编程做应用，接口定义了一个规范，按规范使用才可发挥接口提供的功能。如System.out.println(“Hello World”);

System这个类，定义了一个静态属性out，才能访问out

System
·out
·方法()

而out属于PrintStream类型，而PrintStream类定义了打印字符串的方法。

PrintStream
·属性
·println(String)

故通过out调用该方法，传入一个字符串参数“Hello World”。

作用：帮助用户了解java系统类库，解释类库中每个类的用途，它有什么属性、方法、参数，返回什么类型等等。

其中：

“Nested Class Summary”嵌套类总结，列出嵌套在类里面的类，即所谓的内部类。

“Field Summary”属性总结，列出这个类的所有属性

“Method Summary”方法总结，列出这个类的所有方法

“Field Details”属性详情，是对每个属性的详细说明

“See Also”参考资料，列出跟某属性有关的类及其方法

试题B：卡片

【问题描述】小蓝有很多数字卡片，每张卡片上都是数字0到9。

小蓝准备用这些卡片来拼一些数，他想从1开始拼出正整数，每拼一个，就保存起来，卡片就不能用来拼其它数了。

小蓝想知道自己能从1拼到多少。

例如，当小蓝有20张卡片，其中0到9各3张，则小蓝可以拼出1到10，但是拼11时卡片1已经只有一张了，不够拼出11。

现在小蓝手里有0到9的卡片各2021张，共20210张，请问小蓝可以从1拼到多少？

知识点

1.类的成员的四种访问控制方式

类的成员包括：成员域、成员方法、构造方法等。

公共模式（public）

保护模式（protected）

默认模式（default）

私有模式（private）

2.修饰词static

类的成员域和成员方法可以具有static（静态属性）

3./=

tmp /= 10等价于

tmp = tmp / 10，即把tmp / 10后的值再赋值给tmp

4./和%

运算符/：除法运算符，且运算结果遵从向下取整

运算符%：模运算

试题A：星期计算

【问题描述】【填空题】

已知今天是星期六，请问$20^22$天后是星期几？

注意用数字1到7表示星期一到星期日。

【解】

即先乘再取模 = 先取模再乘

则答案为星期日

试题C：字符统计

【问题描述】

给定一个只包含大写字母的字符串S，请你输出其中出现次数最多的字母。

如果有多个字母均出现了最多次，按字母表顺序依次输出所有这些字母。

【输入格式】

一个只包含大写字母的字符串S.

【输出格式】

若干个大写字母，代表答案。

知识点

1.new关键字的使用

java中任何变量在被使用前都必须先设置初值，利用构造方法为类的成员变量赋初值。构造方法是一种特殊的成员方法，特殊性有：

构造方法名必须与其类名相同；

构造方法是在创建一个对象时使用new进行调用，构造方法的作用是初始化对象，在创建对象时自动执行。

使用“new+构造方法“创建对象：

new关键字创建出一个对象之后，会把该对象在内存中的地址返回，即通过地址就可以找到这个对象。

2.java中Arrays类的用法：操作数组

具有以下功能：

1）数组赋值：Arrays.fill()

2）数组排序：通过sort方法，对指定的int型数组按数字升序排列。

3）比较数组：通过equals方法比较数组中元素值是否相等

4）查找数组元素：通过binarySearch方法能对排序好的数组进行二分查找法操作

5）打印数组：Arrays.toString()

3.java字符串函数

1）equals

判断两个字符串的内容是否相同：s1.equals(s2)，返回值为bool类型（true或false）

2）compareTo

比较两个字符串的大小：s1.compareTo(s2)，比较依据是Unicode编码

若s1>s2，返回值为正；若s1=s2，返回值为0；若s1＜s2，返回值为负

3）length

判断字符串的长度：s1.length()

4）charAt

访问字符串内的单个字符：charAt(index)

index的范围是0~length-1，若index超过length-1会返回error

5）substring

得到字符串的子串：substring()

6）indexOf

判断字符串内是否存在某个字符或字符串：s1.indexOf()

而s1.indexOf(c,n)则是从n号位置开始寻找c

若存在，返回值为该字符或字符串首元素的索引；不存在则返回-1

7）toCharArray

将字符串转换为字符数组，返回值为字符数组s1.toCharArray()

4.continue语句

当满足某条件时，执行continue语句，continue语句会终止本次循环。即循环体中continue之后的语句将不再执行。接着进行下次循环。

5.Eclipse调试

F5跳入方法

F6向下逐行调试

F7跳出方法

F8直接跳转到下一个断点
